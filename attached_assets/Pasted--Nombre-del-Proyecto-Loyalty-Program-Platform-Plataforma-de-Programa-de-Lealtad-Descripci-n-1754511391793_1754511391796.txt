📛 Nombre del Proyecto:
Loyalty Program Platform (Plataforma de Programa de Lealtad)

📌 Descripción General:
Crear una plataforma web autoadministrable en PHP, jQuery, HTML y CSS (sin frameworks modernos), para la gestión de un programa de lealtad de fabricantes y distribuidores de productos (hardware y software). El sistema permitirá a los usuarios acumular puntos por ventas realizadas, redimir premios y visualizar su historial. Además, los administradores podrán gestionar usuarios, deals, reportes y campañas.

🎯 Objetivo Principal:
Recompensar y fidelizar a vendedores, distribuidores y partners a través de un sistema de puntos basado en sus ventas, con funcionalidades administrativas y reportes por país, usuario, nivel de partner, y más.

🌐 Tecnologías a Utilizar:

Frontend: HTML5, CSS3, jQuery (para AJAX e interacciones dinámicas)

Backend: PHP puro (sin framework), estructurado en archivos independientes (MVC opcional)

Base de Datos: MySQL o MariaDB

Gestión de sesiones: PHP sessions (login/logout con permisos por rol)

Internacionalización: Soporte multilenguaje (Español, Inglés, Portugués), usando archivos de idioma .php o .json
